## התחלה מהירה

- דרישות מקדימות: `git`, `docker`, `kubectl` (כדי לעבוד עם התרגילים)
- התקנה:
	1. העתק את המאגר: `git clone <repo-url>`
	2. היכנס לתיקייה: `cd devops-course`
- הרצת תרגיל לדוגמה (Docker):
	1. עבור לתיקיית התרגיל הרלוונטית.
	2. בנה את התמונה: `docker build -t my-exercise .`
	3. הרץ את המכולה: `docker run --rm my-exercise`

אם תרצה, אני יכול להוסיף מדורים נוספים (תרגילים מפורטים, הוראות CI/CD, תיעוד Kubernetes).
